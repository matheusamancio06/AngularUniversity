Sobre o Projeto

O AngularUniversity é uma aplicação web responsiva desenvolvida com Angular 19 e TypeScript, que permite consultar universidades de qualquer país do mundo através da Universities API (Hipolabs).

O projeto foi desenvolvido como atividade final complementar da disciplina de Desenvolvimento de Aplicativos, aplicando os conceitos de componentes, serviços, interfaces, roteamento, HttpClient, data binding, diretivas e armazenamento local.
